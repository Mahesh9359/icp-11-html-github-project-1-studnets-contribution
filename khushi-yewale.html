Tags i've learn in html

  • p tag defines paragraph tag.

  • br tag defines Inserts a line break (self-closing).
  • hr tag defines Inserts a horizontal line (self-closing).
  • b tag defines Makes text bold.
  • i tag defines Makes text italic.
  • u tag defines Underlines text.
  • sub tag defines Subscript text.
  • sup tag defines Superscript text
  • ul tag defines Unordered (bulleted) list.
  • ol tag defines Ordered (numbered) list.
  • table tag Defines a table.
  • tr tag defines a row in a table.
