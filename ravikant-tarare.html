HTML


HTML stands for Hyper Text Markup Language.
HTML used to define the structure and content in webpages. HTML is like a skeleton. HTML uses tags and attributes to define structure & content in webpages.

TAGS & ATTRIBUTES:

Empty Tag:
It is also called as Self-Closing Tag.
eg: Break Row tag.

Container tag:
It is also called as Paired tag. It has opening & closing tags. We write content in between opening tag & closing tag.
eg: Bold tag.
